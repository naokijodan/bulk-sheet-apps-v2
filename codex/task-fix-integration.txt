### 修正1: AIExtractor.gs — isRequiredField_ の判定ロジック

ファイル: /Users/naokijodan/Desktop/ツール開発/一括シートApps_v3/ItemSpecifics/AIExtractor.gs

現在のコード（バグ）:
```javascript
function isRequiredField_(fieldDef) {
  if (!fieldDef) return false;
  if (fieldDef.required === true) return true;
  var p = (fieldDef.priority || '').toLowerCase();
  return p === 'required' || p === 'high' || p === 'must';
}
```

問題: `fieldDef.priority` は数値（1, 2, 3...）なので `.toLowerCase()` でエラーになる。
必須判定は `fieldDef.type` が 'required' かどうかで判定すべき。

修正後:
```javascript
function isRequiredField_(fieldDef) {
  if (!fieldDef) return false;
  var t = String(fieldDef.type || '').toLowerCase();
  return t === 'required';
}
```

### 修正4: ItemSpecifics.gs — getActiveISSettings_ のプロパティ名マッピング

ファイル: /Users/naokijodan/Desktop/ツール開発/一括シートApps_v3/ItemSpecifics/ItemSpecifics.gs

現在のコード:
```javascript
function getActiveISSettings_() {
  var settings = {};
  try {
    if (typeof getISSettings === 'function') {
      var s = getISSettings();
      if (s && typeof s === 'object') {
        settings = s;
      }
    }
  } catch (e) {
    Logger.log('[getActiveISSettings_] error: ' + e);
  }
  // デフォルト補完
  if (!settings.targetSheetName) { settings.targetSheetName = '出品2'; }
  if (!settings.dataStartRow) { settings.dataStartRow = 3; }
  return settings;
}
```

問題: Config_IS.gs の getISSettings() は以下のキー名を使う:
- SHEET_NAME ('出品2')
- HEADER_ROW (2)
- DATA_START_ROW (3)
- DICTIONARY_SHEET_ID
- OPENAI_API_KEY

しかし ItemSpecifics.gs では別のキー名を期待している:
- targetSheetName
- dataStartRow
- dictionarySpreadsheetId
- apiKey

修正: Config_IS.gs のキー名に統一する。

```javascript
function getActiveISSettings_() {
  var settings = {};
  try {
    if (typeof getISSettings === 'function') {
      var s = getISSettings();
      if (s && typeof s === 'object') {
        settings = s;
      }
    }
  } catch (e) {
    Logger.log('[getActiveISSettings_] error: ' + e);
  }
  // Config_IS.gs のキー名でデフォルト補完
  if (!settings.SHEET_NAME) { settings.SHEET_NAME = '出品2'; }
  if (!settings.DATA_START_ROW) { settings.DATA_START_ROW = 3; }
  // 互換エイリアス
  settings.targetSheetName = settings.SHEET_NAME;
  settings.dataStartRow = settings.DATA_START_ROW;
  settings.dictionarySpreadsheetId = settings.DICTIONARY_SHEET_ID || '';
  settings.apiKey = settings.OPENAI_API_KEY || '';
  return settings;
}
```
