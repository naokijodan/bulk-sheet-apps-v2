# Ver.44 更新内容 - 関税率調整方式

## 概要

Ver.44では、eBayのDDP（関税込み価格）に対する手数料課金問題に対応するため、**関税率調整方式**を導入しました。

Ver.43で採用したDDP基準の価格計算式から、**旧DDU基準の価格計算式に戻し**、代わりに**調整関税率**を使用して関税額を計算する方式に変更しました。

---

## バージョン比較

### Ver.42以前（旧式）

**販売価格の計算：DDU基準**
```
販売価格 = (仕入れ + 送料 + 利益) / (1 - 手数料 - 広告 - Payoneer) / 為替
```

**想定関税の計算：実際の関税率**
```
想定関税 = 販売価格 × 実際の関税率 × (1 + 処理手数料率) + ...
```

**問題点：**
- eBayがDDP価格（販売価格＋関税）に対して手数料を課すようになった
- 旧式では関税額分の手数料が考慮されていない
- 高額商品ほど利益が目減りする

---

### Ver.43（DDP手数料対応版）

**販売価格の計算：DDP基準に変更**
```
販売価格 = (仕入れ + 送料 + 利益) / ((1 - Payoneer) - (1 + 関税率×(1+処理手数料率)) × (手数料 + 広告)) / 為替
```

**問題点：**
- 販売価格自体が上がる → 競争力が下がる
- 計算式が複雑
- 既存出品の価格修正が必要

---

### Ver.44（関税率調整方式）★今回

**販売価格の計算：旧DDU基準に戻す**
```
販売価格 = (仕入れ + 送料 + 利益) / (1 - 手数料 - 広告 - Payoneer) / 為替
```

**想定関税の計算：調整関税率を使用**
```
調整関税率 = 実際の関税率 / (1 - 手数料率 - 広告費率) × 1.03
想定関税 = 販売価格 × 調整関税率 × (1 + 処理手数料率) + ...
```

---

## Ver.44のメリット

| 項目 | Ver.43 | Ver.44 |
|------|--------|--------|
| 販売価格 | 上昇（競争力低下） | 旧式と同じ（競争力維持） |
| 計算式 | 複雑 | シンプル（旧式） |
| 利益確保 | 販売価格で調整 | 関税額で調整 |
| 端数誤差対策 | なし | 3%安全係数で対応 |

---

## 調整関税率の仕組み

### 考え方

eBayはDDP価格（販売価格＋関税）に対して手数料を課します。

つまり、関税額にも手数料がかかるため、その分を関税額に上乗せしてバイヤーから徴収する必要があります。

### 計算式

```
調整関税率 = 実際の関税率 / (1 - 手数料率 - 広告費率) × 1.03
```

### 計算例

- 実際の関税率: 15%
- 手数料率: 18%
- 広告費率: 5%

```
調整関税率 = 15% / (1 - 0.18 - 0.05) × 1.03
           = 15% / 0.77 × 1.03
           = 19.48% × 1.03
           = 20.07%
```

この調整関税率（20.07%）で計算した関税額をバイヤーから徴収することで、DDP手数料分を補填します。

## セル配置の変更

### 新規追加セル

| セル | 内容 | 説明 |
|------|------|------|
| AA2 | 実際の関税率 | ユーザーが入力（例: 0.15 = 15%） |
| AF2 | 調整関税率 | 自動計算（式: `=$AA$2/(1-$F$1-$F$2)*1.03`） |
| AH列 | 実際の関税額 | 参考表示用。AA2（実際の関税率）で計算 |

### 既存セルの変更

| セル | 変更内容 |
|------|----------|
| AD列（想定関税） | AF2（調整関税率）を使用するように変更 |

## 既存出品への対応

### 販売価格
- **修正不要** - Ver.42以前と同じ計算式のため

### シッピングポリシー
- **レガシー再計算が必要**
- 想定関税額が変わるため、シッピングポリシーの判定結果が変わる可能性あり
- 一括シートから「レガシー再計算」を実行してください

---

## 利益計算の確認方法

### 想定関税額 vs 実際の関税額

- **AD列（想定関税）**: バイヤーから徴収する関税額（調整関税率で計算）
- **AH列（実際の関税額）**: 実際に支払う関税額（実際の関税率で計算）

この差額が、DDP手数料を補填するための上乗せ分です。

### 計算例

販売価格: $152.69

| 項目 | 金額 |
|------|------|
| 想定関税（AD列） | $31.28（調整関税率20.07%） |
| 実際の関税額（AH列） | $23.39（実際の関税率15%） |
| 差額（上乗せ分） | $7.89 |

この$7.89（約1,223円）が、DDP手数料分を補填するためにバイヤーから追加徴収される金額です。

## 注意事項

1. **AA2（実際の関税率）の設定を忘れずに**
   - デフォルトは15%（0.15）
   - 商品カテゴリーに応じて変更してください

2. **初期設定または式の再出力が必要**
   - 新しい計算式を反映するため、初期設定を再実行するか、「計算式を再出力」を実行してください

3. **3%の安全係数について**
   - 端数処理や為替変動による誤差を吸収するため、調整関税率に3%の安全係数を掛けています
   - これにより、利益が目減りするリスクを軽減しています
