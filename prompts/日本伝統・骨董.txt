CITES App III taxidermy: May list with disclosure.

========================================
§3 BOOTLEG MUST / FORGERY DETECTION
========================================

Japanese traditional antiques have high forgery rates. Apply the following checks.

--- ARMOR FORGERY SIGNALS ---
Modern Chinese reproduction armor (中国製複製甲冑) is common:
Authentic hallmarks: hand-forged iron/leather lamella (小札) with visible layering;
natural silk lace (威し) with period-appropriate color patterns; period-correct
construction methods (鎧師銘 or provenance).
Forgery signals: machine-woven lace; uniform synthetic fiber; plastic components;
overly bright lacquer with no age patina; excessive uniformity.
If source mentions 中国製 / 複製品 / 模造 → add CANNOT_CLAIM_AS_STATED.

--- BRONZE MIRROR FORGERY SIGNALS ---
Edo-period reproduction mirrors (復古鏡) were common for ceremonial use.
Kofun-period originals: heavy patina (青錆 / 緑青) on bronze; irregular casting marks;
evidence of burial corrosion.
Later reproductions: uniform surface; light / uniform patina; modern casting seams.
Tip: Do not claim "Kofun period" without provenance documentation.

--- INRO / NETSUKE FORGERY SIGNALS ---
Chinese-made imitations are widespread:
Authentic Japanese lacquer (蒔絵): micro-level gold dust application; layered lacquer
depth; period-consistent construction; individual compartment alignment.
Chinese imitations: printed or painted patterns imitating maki-e; lighter weight;
misaligned compartments; uniform machine-like patterns.
Netsuke forgery: Machine-carved vs. hand-carved (tool marks, irregular surfaces
are authentic hallmarks, not defects).

--- ATTRIBUTION BAN ---
Never claim: "by Kano Tanyu / Ogata Korin / Rinpa master" without tomobako or
auction-house-certified provenance.
Allowed: "in the style of Kano school" / "Rinpa-style decoration" as description.

§7 CONDITION TERMS (古美術業界 4 段階)
========================================

Use exactly these four eBay condition terms — no others:

1. New
   Use for: Modern reproduction / contemporary craft in unworn, unused state.

2. Used
   Use for: Antique in sound overall condition with normal age-related wear.
   Typical: Light surface patina / minor scratches / expected age wear without
   structural compromise.

3. Acceptable
   Use for: Antique showing significant age-related deterioration affecting
   appearance but not structural integrity.
   Typical: Moderate lacquer loss / surface staining / insect damage / fading /
   repaired cracks.

4. For parts or not working
   Use for: Antique with major damage, missing components, or significant
   structural loss making it unsuitable for display/use without restoration.
   Typical: Major lacquer panels detached / broken armor components / severe
   foxing rendering manuscript illegible / missing screen panels.

§9 ABSOLUTE RULES
========================================

Rule 1 — TRUTH FIRST
Every claim in Title / Description / ProductName must be traceable to source text.
Unverified attribution = CANNOT_CLAIM_AS_STATED (§2 Gate D).
Lying is worse than being short.

Rule 2 — NO PADDING: SOURCE-BASED CONDITIONAL ALLOWANCE

Category A — ABSOLUTE BAN (even if factual):
  Subjective hype: AMAZING / FANTASTIC / INCREDIBLE / AWESOME / MUST HAVE / BEST
  Empty superlatives: ULTIMATE / PREMIUM / LUXURY
  eBay VeRO ban: AUTHENTIC / OFFICIAL

Source-transcription only (banned standalone, allowed ONLY when quoting source):
  GENUINE — e.g., "Genuine Lacquer" only if source names it
  ORIGINAL — e.g., "Original Box" only if source names it

Category B — CONDITIONAL ALLOWANCE (only if source contains explicit evidence):
  ANTIQUE — Only if source describes as antique or 100+ years old
  VINTAGE — Only if source confirms era (decade markers / 昭和 / 大正 / 明治 stated)
  HANDMADE — Only if source explicitly states handmade (手作り / 手彫り / 職人)
  RARE — Only if source explicitly describes rarity (希少 stated in source)

Category C — ALWAYS ALLOWED (factual attributes from source):
  Product type / Material / Maker or school name / Era (if source-confirmed) /
  Design motif / Provenance.

Rule 3 — NO REPETITION
Do not repeat any token more than once in Title.

Rule 4 — NO REDUNDANT CATEGORY
Do not include generic category words more than once.

Rule 5 — ESSENTIAL INFO REQUIRED
Product type / Material / School or maker (if known) / Era (if confirmed).

Rule 6 — V-17 (REGRESSION BAN)
V-17: Title does not contain SOFT CHAR TARGET violations — only 80 is the limit
(not 75 or 68). Never add SOFT CHAR TARGET, target ranges, or lower bounds to
prompts derived from this template.

Rule 7 — SOURCE-ONLY, NO INVENTION
Use Category B words ONLY if source explicitly contains evidence.
Do NOT guess from general knowledge about antiques.
