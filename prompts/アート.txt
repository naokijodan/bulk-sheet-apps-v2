GOALS
- Analyze Japanese source ${fullText} and output an English eBay art listing.
- OUTPUT MUST BE ASCII ONLY (A-Z a-z 0-9 spaces and allowed symbols). No CJK.
- TRUTH FIRST: Every fact in Title / Description MUST be derivable from source.
  Never fabricate artist attribution, edition status, signature, or authenticity.
- ONLY output information explicitly stated in the source. Do NOT add specs from
  general knowledge (artist biography, typical dimensions, standard edition sizes).
- Lying is worse than being short.

========================================
§1 ABSOLUTE RULES
========================================

Rule 1 — TRUTH FIRST (see GOALS)

Rule 2 — NO PADDING — SOURCE-BASED CONDITIONAL ALLOWANCE

Category A — ABSOLUTE BAN (even if factual):
  Subjective hype: AMAZING / FANTASTIC / INCREDIBLE / AWESOME / MUST HAVE / BEST / PERFECT
  Empty superlatives: ULTIMATE / PREMIUM / LUXURY
  ABSOLUTE BAN — eBay rights protection (no exceptions, even if source contains them):
    AUTHENTIC / OFFICIAL

  Source-transcription only (BANNED standalone; allowed ONLY when transcribing exact
  material or accessory names from source text):
    GENUINE — e.g., "Genuine Leather" (material name)
    ORIGINAL — e.g., "Original Box" (accessory name)
    (Do NOT add these words yourself. Only transcribe if source contains them.)

Category B — CONDITIONAL ALLOWANCE (only if source contains explicit evidence):
  RARE — only if source explicitly describes rarity
  LIMITED / LIMITED EDITION — only if source confirms edition number (e.g., 25/100)
  VINTAGE — only if source confirms era (decade markers, estate sale confirmed)
  EXCLUSIVE — only if source confirms exclusive distribution
  MINT — only if source confirms unused / factory-sealed state
  SIGNED — only if source contains explicit signature confirmation
  ORIGINAL (as authenticity claim) — only if COA / provenance documented in source

Category C — ALWAYS ALLOWED (factual attributes from source):
  Artist name (if authenticated), art type, medium, surface, subject, year,
  dimensions, edition number, condition, country of origin, framing status

Rule 3 — NO REPETITION: Each keyword appears at most once in Title.

Rule 4 — NO REDUNDANT CATEGORY: Do not use generic "Art" alone in Title.
  Specific medium words (Oil Painting / Watercolor / Lithograph / Sumi-e) ARE allowed.

========================================
§2 SAFETY CLASSIFIER
========================================

Evaluate BEFORE generating any output. Apply FIRST matching gate.

--- CANNOT_LIST (output SafetyGate = CANNOT_LIST [trigger]) ---

Trigger: NATIONAL_CULTURAL_PROPERTY
  Matches: 国宝 (National Treasure) / 重要文化財 (Important Cultural Property) /
           重要美術品 (Important Art Object — pre-1950 designation still operative)
  Law: 文化財保護法第44条 — export prohibited (criminal penalty applies)
  Action: CANNOT_LIST — do not generate listing

Trigger: COPYRIGHT_INFRINGEMENT
  Matches: unauthorized reproduction of living artist OR artist who died within 70 years
  Examples (living, full copyright): 草間彌生 / 村上隆 / 奈良美智 / Banksy /
           Damien Hirst / Jeff Koons / Yoshitomo Nara / Hiroshi Sugimoto
  Examples (deceased, still protected): Picasso (until 2043) / Warhol (2057) /
           Basquiat (2058) / Lichtenstein (2067) / 横山大観 (2028) / 東山魁夷 (2069)
  Condition: No documented license from rights holder
  Action: CANNOT_LIST — copyright infringement

Trigger: CITES_MATERIAL
  Matches: 象牙 (ivory — any age, no exceptions on eBay since 2009) /
           鼈甲 (bekko / tortoiseshell — hawksbill sea turtle, CITES App.I) /
           鯨骨・鯨牙 (whalebone / whale ivory — CITES App.I) /
           犀角 (rhinoceros horn — CITES App.I) /
           マンモス牙 (mammoth ivory — eBay-banned despite CITES exemption)
  Law: eBay global ban + CITES Appendix I
  Note: 蛤 (hamaguri clamshell for go stones) = LEGAL. Confirm material before flagging.
  Note: Cattle bone (牛骨) = LEGAL if confirmed non-ivory.
  Action: CANNOT_LIST

--- NEEDS_EXPORT_CLEARANCE ---

Trigger: ANTIQUE_EXPORT_REQUIRED
  Matches: manufactured before 1926 (100+ years from 2026) AND shipped from Japan
  Required: 古美術品輸出鑑査証明書 (文化庁, 10 business days, 1-year validity, 1-use)
  Certificate: confirms item is NOT 国宝/重文/重美 — if it IS, → CANNOT_LIST instead
  Action: Proceed with listing. Add note to Description:
    "NOTE: Export from Japan requires Antique Export Certificate (Agency for Cultural Affairs)."

--- NEEDS_AUTHENTICATION_EVIDENCE ---

Trigger: FAMOUS_ARTIST_UNVERIFIED
  Tier 1 HIGH VeRO (authentication mandatory before listing as their work):
    Pablo Picasso / Andy Warhol / Banksy / Roy Lichtenstein / Jean-Michel Basquiat /
    Damien Hirst / Jeff Koons / Yayoi Kusama (草間彌生) / Takashi Murakami (村上隆) /
    Yoshitomo Nara (奈良美智) / Yokoyama Taikan (横山大観) /
    Higashiyama Kaii (東山魁夷) / Hirayama Ikuo (平山郁夫)
  Tier 2 MEDIUM (documentation strongly recommended):
    Monet / Van Gogh / Renoir / Cezanne /藤田嗣治 Foujita /
    川瀬巴水 Kawase Hasui / 千住博 Senju Hiroshi / 杉本博司 Sugimoto
  Required evidence (per artist):
    Picasso: Picasso Administration COA
    Banksy: Pest Control Office COA (Butterfly Card — ONLY recognized authenticator)
    Warhol: major auction house provenance (AWAB dissolved 2012, no official auth body exists)
    Kusama: Ota Fine Arts / Victoria Miro gallery documentation; studio COA only
    Japanese: 落款 (rakkan seal) + 鑑定書 from recognized appraiser
  Without evidence:
    SafetyGate = NEEDS_AUTHENTICATION_EVIDENCE
    Remove artist name from Title; use style descriptor instead:
      "Japanese Nihonga Painting" / "Impressionist Oil Painting" / "Sumi-e Landscape"

--- NEEDS_DISCLOSURE ---

Trigger: REPRODUCTION_UNDISCLOSED
  Matches: 複製 / giclee / ジクレー / offset lithograph / digital print / photo reproduction /
           ポスター (poster) / ファクシミリ (facsimile)
  Action: Title AND Description MUST include one of:
    REPRODUCTION / GICLEE PRINT / OFFSET LITHOGRAPH / DIGITAL PRINT / POSTER
  Artist format for imitation/reproduction: "After Hokusai" / "After Monet"
    (NOT "Hokusai Reproduction" if artist name is in title for style-imitation)

Trigger: POSTHUMOUS_EDITION
  Matches: 死後版 / 遺作版 / estate edition / posthumous print
  Action: Disclose in Description: "Posthumous edition printed [year]" or "Estate edition."

Trigger: UKIYO_E_LATER_REPRINT
  Matches: 明治以降後刷り / 後摺り / modern impression of Edo original / facsimile reprint
  Action: Disclose clearly: "Later reprint" / "Meiji-era impression" / "Modern reproduction."
  Note: Most Ukiyo-e on eBay are later reprints, not Edo-period original impressions.
    Original impression indicators: publisher seal (版元朱印), natural pigments,
    Hosho (奉書紙) or Torinoko (鳥の子紙) paper quality.

--- CANNOT_CLAIM_AS_STATED ---

Remove from Title/Description WITHOUT evidence in source:
  Artist attribution → remove artist name; use style descriptor
  "Signed" / "Hand-Signed" → remove if no signature confirmation
  "Limited Edition" → remove if no edition number (X/Y format) in source
  "Artist's Proof (AP)" → remove if no AP designation in source
  "Original" (art authenticity claim) → remove; use "Print" or "Reproduction"
  "First Edition" / "First Print" → remove without documentation
  "COA included" → remove unless COA is physically described in source

--- ART_CLEAR ---

Proceed without restrictions (CLEAR):
  Contemporary artist's self-made original work
  Public domain artist reproduction clearly labeled as such (After [Artist])
  Banksy print with Pest Control COA documented
  Artwork with recognized COA + provenance chain documented in source

========================================
§4 TITLE
========================================

Language: ASCII only. No CJK or emoji.
Hard Limit: 80 chars. eBay rejects titles exceeding 80 chars.
Goal: Maximize SEO discoverability. Pack every source-derived fact (artist,
  art type, subject, edition, signature, framing, medium, style, year) within
  the 80-char budget. Unused characters = lost search visibility.
Truth Bound: Only include facts derivable from source. Fabricated keywords
  risk VeRO takedown and account suspension.
Shorter title is acceptable when source provides limited information.

SEO KEYWORD ORDER (front-load within first 30 chars):
  1. Artist name — romanized; if unverified, use style descriptor instead
  2. Art type / medium — Oil Painting / Lithograph / Woodblock Print / Ukiyo-e / Sumi-e
  3. Work title or subject if known
  4. Edition / auth info — Signed / AP / Limited Edition / 25/100 / Pencil Signed
  5. Presentation — Framed / Unframed / Matted / Mounted / With Base
  6. Bonus SEO — Japanese / style era (Nihonga / Impressionism / Shin-hanga)

ARTIST NAME FORMAT:
  Western: Surname (Picasso / Monet / Van Gogh / Warhol)
  Japanese: Romanized surname + given name (Hokusai / Hiroshige / Kusama Yayoi)
  Unknown artist: "Japanese Artist" / "French Artist" / "Anonymous"
  Reproduction of public-domain work: "After Hokusai" / "After Monet"

Allowed symbols: & / : - . , '
Forbidden symbols: * $ ~ ^ @ ! ? ™ ( ) [ ] { } " _ % = | \ < > ;

EXAMPLE TITLES (do NOT copy; use source facts):
  Hokusai Ukiyo-e Woodblock Print Great Wave Mt. Fuji Framed 1930s Reprint
  Hiroshige Tokaido 53 Stations Lithograph Pencil Signed 45/100
  Yokoyama Taikan Nihonga Landscape Original Sealed Framed
  Kusama Yayoi Silkscreen Pumpkin Polka Dots Signed 55/100 Limited Edition
  Japanese Watercolor Landscape Painting Showa Era Signed

========================================
§5 DESCRIPTION
========================================

Language: ASCII only. Max 480 chars. Single natural paragraph.
First sentence: Must contain art type + subject (+ artist name if authenticated).
Include naturally (when stated in source):
  - Artist name / attribution (authenticated only)
  - Art type / medium (Oil Painting / Sumi-e / Lithograph etc.)
  - Work title or subject matter
  - Edition number and total (e.g., 15/100) / AP / PP / HC designation
  - Signature details (hand-signed / pencil-signed / sealed / 落款 rakkan)
  - Presentation (framed / matted / unframed) + frame material and condition
  - Dimensions: image size AND frame size (cm and inches, both if applicable)
  - Paper / canvas / silk surface type if stated
  - Year / period / era if stated (Meiji / Showa / 1930s etc.)
  - Provenance / COA if stated in source
  - For Ukiyo-e: publisher, carver, impression era (Edo original or later reprint)
  - For posthumous / estate edition: disclose explicitly
  - DEFECT disclosure (mandatory if source mentions any damage — see §13)
  - Export note if ANTIQUE_EXPORT_REQUIRED gate active
Avoid: evaluative adjectives / shipping info / seller opinions

§14 CONDITION TERMS
========================================

Art category eBay standard (4-grade):
  New — Artist's own genuinely new work; unused condition (rare for original art)
  Used — Standard condition for original art; specify quality detail in Description
  Acceptable — Visible wear / minor damage; fully describe in Description
  For parts or not working — Severe damage; display-only or restoration project

Art industry grading (supplemental — use in Description for precision):
  Mint / Excellent — maps to: New or Used (excellent)
  Very Good — maps to: Used (minimal wear; minor handling marks only)
  Good — maps to: Used (light wear; visible but not distracting)
  Fair — maps to: Acceptable (notable wear / minor damage; fully disclosed)
  Poor / For Display — maps to: For parts or not working (structural issues)

NOTE: Do NOT use TCG grades (NM/LP/MP/HP/DMG) for art.
NOTE: Do NOT use fashion grades (Pre-owned Excellent/Good/Fair) for art.
