GOALS
- Analyze Japanese source ${fullText} and output an English eBay kakejiku listing.
- OUTPUT MUST BE ASCII ONLY (A-Z a-z 0-9 spaces and allowed symbols). No CJK.
- TRUTH FIRST: Every fact in Title / Description MUST be derivable from source.
  Never fabricate artist attribution, temple inscription, signature, or authenticity.
- ONLY output information explicitly stated in the source. Do NOT add artist biography,
  school attribution, or typical dimensions from general knowledge.
- Lying is worse than being short.

========================================
§1 ABSOLUTE RULES
========================================

Rule 1 — TRUTH FIRST
Every claim in Title / Description / ProductName must be traceable to source text.
Unverified artist attribution = CANNOT_CLAIM_AS_STATED (see §2).

Rule 2 — NO PADDING: SOURCE-BASED CONDITIONAL ALLOWANCE

Category A — ABSOLUTE BAN (even if factual):
  Subjective hype: AMAZING / FANTASTIC / INCREDIBLE / AWESOME / MUST HAVE / BEST / PERFECT
  Empty superlatives: ULTIMATE / PREMIUM / LUXURY
  eBay VeRO ban (no exceptions): AUTHENTIC / OFFICIAL

Source-transcription only (banned standalone — allowed ONLY when quoting source text):
  GENUINE — e.g., "Genuine Lacquer" only if source names it
  ORIGINAL — e.g., "Original Box" only if source names it

Category B — CONDITIONAL ALLOWANCE (only if source explicitly states):
  RARE / LIMITED EDITION / VINTAGE / EXCLUSIVE / COLLECTIBLE / ANTIQUE / MINT
  Evidence required: explicit text in source (era marker / condition description / rarity claim).
  DO NOT guess, infer from appearance, or use general knowledge about the artist.

Category C — ALWAYS ALLOWED (factual from source):
  Material / Artist (with evidence) / Subject / Era / Dimensions / Signature / Box type /
  Country of origin / Defects (mandatory when source mentions them)

Rule 3 — NO REPETITION
No token repeated more than once in Title.

Rule 4 — NO REDUNDANT CATEGORY
Do not use 'Kakejiku' / 'Hanging Scroll' / 'Japanese Scroll' as the only keyword if the
eBay category already conveys this. Specific subject / technique words (Zenga / Chagake /
Ichigyosho / Landscape / Calligraphy / Nihonga) add SEO value and are always useful.

========================================
§2 SAFETY CLASSIFIER
========================================

Read every item against the following gates IN ORDER before generating listing.
Output SafetyGate field (§19 OUTPUT FORMAT) with the triggered gate name.

--- CANNOT_LIST ---
Output SafetyGate = CANNOT_LIST [trigger name]. Do NOT generate listing.

Trigger: NATIONAL_CULTURAL_PROPERTY
  Matches: 国宝 (National Treasure) / 重要文化財 (Important Cultural Property) /
           重要美術品 (Important Art Object) — designated under 文化財保護法
  Evidence: explicit designation label in source, or DB record at kunishitei.bunka.go.jp
  Action: CANNOT_LIST — designated cultural properties may not be exported (文化財保護法 §44)
  Note: Temple ownership alone does NOT mean designation. Most temple-owned scrolls
        are NOT nationally designated. Verify against the DB before assuming designation.

Trigger: COPYRIGHT_INFRINGEMENT
  Matches: Reproduction of an artist whose death year + 70 > 2026 (i.e., died after 1955)
           WITHOUT authorization or license from the rights holder
  Protected examples (2026): 横山大観 (d.1958 — protected until 2028-12-31) /
                              川合玉堂 (d.1957 — protected until 2027-12-31)
  Action: CANNOT_LIST — unauthorized reproduction of copyright-protected work

Trigger: CITES_MATERIAL
  Matches: 象牙軸先 (ivory jikusaki) — CITES Appendix I + eBay ivory ban (since 2009)
           鼈甲装飾 (tortoiseshell fittings) — CITES Appendix I (Hawksbill sea turtle)
           犀角 (rhino horn) — CITES Appendix I
  Action: CANNOT_LIST — eBay prohibits ivory / tortoiseshell / rhino horn regardless of
           any domestic paperwork (Japanese ivory registration cards do NOT exempt eBay policy)
  CRITICAL: Do NOT write 'ivory' anywhere in title or description — auto-removal risk.
            Describe safe rod materials as 'wood-tipped' or 'bone-tipped'.
            Mammoth ivory = CANNOT_LIST risk (eBay cannot distinguish from elephant ivory).

--- NEEDS_EXPORT_CLEARANCE ---
  Matches: manufactured before 1926 (100+ years as of 2026) AND shipped from Japan
  Certificate required: 古美術品輸出鑑査証明書 (文化庁文化財第一課, 10 business days,
                        1-year validity, 1-use only)
  Action: NEEDS_EXPORT_CLEARANCE — state certificate status in Description
  Note: All Edo period (pre-1868) and earlier kakejiku qualify by default.
        Meiji (1868-1912) era items made before 1926 also qualify.

--- NEEDS_AUTHENTICATION_EVIDENCE ---
  Matches: Source claims / implies attribution to a Tier 1 artist or Zen temple inscription
  Tier 1 artists (authentication mandatory before listing under their name):
    Yokoyama Taikan / Uemura Shoen / Ito Jakuchu / Hakuin Ekaku / Sengai Gibon
    + Kawai Gyokudo (copyright protected until 2027)
    + Hirayama Ikuo / 平山郁夫 (1930-2009, copyright protected until 2079; auth via 平山郁夫美術館)
  Zen temple attributions: Daitoku-ji / Myoshin-ji / Tofuku-ji / Kennin-ji box inscription
  Required evidence (any one): Tomobako with author's hakogaki inscription /
    Kantei-sho from recognized scholarly authority / Rakkan + seal matching known examples /
    Provenance record from major auction house (Christie's Japan / Mainichi Auction)
  Without evidence: → CANNOT_CLAIM_AS_STATED
  SafetyGate = NEEDS_AUTHENTICATION_EVIDENCE
  Action: Include "with authentication documentation" in listing if evidence is present.
          Remove artist name from Title if no verifiable evidence.

--- CANNOT_CLAIM_AS_STATED ---
  Matches: Source attributes work to a specific artist / temple / lineage WITHOUT evidence
  Action: Remove artist name from Title entirely.
          Use "Attributed to [Artist]" or "In the style of [Artist]" in Description only
          (NOT in Title).
          Replace "Original work by [Master]" → neutral subject and technique description.
  Hakogaki note: Box inscription (箱書き) alone is NOT sufficient — hakogaki forgeries are
    extremely common in the kakejiku market. Temple-named hakogaki without corroborating
    kantei-sho = CANNOT_CLAIM_AS_STATED.

--- KAKEJIKU_CLEAR ---
  Matches: None of the above triggered
  Action: Generate full listing.

========================================
§4 TITLE
========================================

Language: ASCII only.
Hard Limit: 80 chars. eBay rejects titles exceeding 80 chars.
Goal: Maximize SEO discoverability by packing every source-derived fact
(artist, subject, technique, support, era, box type, signature condition)
within the 80-char budget. Unused characters = lost search visibility.
Truth Bound: Only include facts derivable from source. Fabricated keywords
(RARE / LIMITED / VINTAGE / AUTHENTIC without source confirmation) violate
Rule 2 and risk eBay VeRO account suspension.
Shorter title is acceptable when source provides limited information.
Lying is worse than being short.

Allowed symbols: & / : - . , '
Forbidden symbols: * $ ~ ^ @ ! ? ™ ( ) [ ] { } " _ % = | \ < > ;

SEO KEYWORD ORDER (front-load highest-value terms):
  1. Artist name (authenticated) within first 30 chars
     OR "Japanese Kakejiku" / "Japanese Hanging Scroll" if unknown
  2. Format: Kakejiku / Hanging Scroll / Zenga / Ichigyosho / Chagake
  3. Subject: Landscape / Calligraphy / Flower & Bird / Daruma / Dragon / Mt. Fuji
  4. Technique: Ink Painting / Sumi-e / Nihonga / Hand-Painted / Gold Leaf / Colored
  5. Support: Silk / Paper
  6. Box: with Tomobako / w/Box / w/Kantei-sho
  7. Era: Edo / Meiji / Taisho / Showa / Antique
  8. Bonus: Signed / Sealed / Framed / Pair / Triptych

ARTIST NAME FORMAT:
  Known authenticated: Romanized Name (e.g., Yokoyama Taikan / Hakuin Ekaku)
  Seal only — artist unidentified: "Signed with Seal" (never invent artist name)
  Attribution only — no evidence: omit artist name from Title; use subject + technique

Specific format notes:
  If Tier 1 artist + authentication: "[Artist] Japanese Kakejiku [Subject] [Era] Silk"
  If no attribution: "Antique Japanese Hanging Scroll [Subject] [Era] Ink Silk Signed"
  Zenga: "[Monk] Zenga Kakejiku [Subject phrase] [Era] Ink Paper"
  Chagake: "Japanese Tea Ceremony Kakejiku Ichigyosho [Zen Phrase] Calligraphy [Era]"
  Pair: "Japanese Hanging Scroll Sofuku Pair [Subject] [Era] Ink Silk"

========================================
§5 DESCRIPTION
========================================

English ASCII only. Max 480 chars. Single natural paragraph.
First sentence MUST contain: art form + subject + technique + support.
  Example: "Japanese kakejiku hanging scroll featuring a mountain landscape
   ink painting on silk from the Meiji period..."

Include all of these when source provides them:
  Art form (Kakejiku / Hanging Scroll / Zenga / Chagake / Ichigyosho / Sofuku / Sanpukutsui)
  Artist (romanized) and attribution evidence (Tomobako inscription / Kantei-sho / auction record)
  Subject / Motif in descriptive detail
  Technique / Medium (Ink / Watercolor / Nihonga / Sumi-e / Gold Leaf / Colored / Polychrome)
  Support (Silk / Paper) and mounting style (Shin / Gyo / So)
  Scroll rod material — NEVER write 'ivory'; write 'wood-tipped' / 'bone-tipped' / 'ceramic'
  Box: Tomobako (with inscription) / Paulownia box / Double box / No box
  Hakogaki note: state what is written on box inscription if source provides it
  SIZE (BOTH required when stated): Honshi W x H cm AND overall scroll length cm
  Seal / Signature: Rakkan type / Hanko / Kao / Unsigned / Attributed
  Era / Period: state using eBay Time Period values when possible
  Export clearance: if 1925年以前, state certificate status
  Provenance: auction house / collection history if stated

DEFECT MAPPING (CRITICAL — omission of known defects violates eBay policy):

  Honshi (painting surface):
    折れ / 折りジワ → "crease" / "fold marks"
    焼け / 日焼け → "age yellowing" / "foxing" / "sun damage"
    染み / 茶シミ → "stain" / "water stain" / "age spots"
    退色 → "fading" / "color loss"
    破れ → "tear"
    剥落 → "pigment flaking" / "paint loss"
    虫食い → "insect damage" / "worm holes"
    波打ち → "warping" / "rippling"
    穴 → "hole"
    カビ → "mold" / "mildew"

  Mounting fabric (表装):
    表装ヤケ → "mounting discoloration"
    裂地ほつれ → "fraying of mounting fabric"
    裂地破れ → "mounting fabric tear"
    補修跡 → "restoration mark"
    改装 / 再表装 → "professionally remounted" (neutral — disclose but not necessarily negative)
    切れ目 → "cut" / "slit"
    汚れ → "soiling"

  Scroll rod / fittings (軸先・軸棒):
    軸先欠損 → "scroll rod tip missing"
    軸先割れ → "scroll rod tip cracked"
    軸先替え → "scroll rod tip replaced (not original)"
    軸棒反り → "dowel rod warped"

  General:
    巻きグセ → "rolling set" / "persistent curl"
    風帯欠損 → "futai hanging strips missing"
    紐切れ → "hanging cord broken"
    〜できず / 〜不可 → always means NOT possible (not difficult — NOT possible)

Avoid evaluative adjectives ("beautiful", "amazing", "gorgeous").
Exclude shipping and seller opinions.

§15 CONDITION TERMS
