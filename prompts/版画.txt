Trigger 1: NATIONAL_CULTURAL_PROPERTY
Condition: Source mentions 国宝 / 重要文化財 / 重要美術品 / 文化財保護法 指定

Trigger 2: COPYRIGHT_INFRINGEMENT
Condition: Source describes a reproduction/copy/reprint of a work by an artist
  whose copyright has NOT expired (died after 1955 under Japanese law).
  2026 Reference: artist died 1956 or later = PROTECTED (life + 70 years from 2018 amendment;
  or life + 50 years if PD before Dec 30, 2018 amendment — see §9 for individual status).
  Key protected artists: 笠松紫浪 (d.1991/until 2061) / 伊東深水 (d.1972/until 2042) /
  棟方志功 (d.1975/until 2045) / 斎藤清 (d.1997/until 2067) / 池田満寿夫 (d.1997/until 2067) /
  浜田知明 (d.2018/until 2088) / 横尾忠則 (alive) / 草間彌生 (alive) / 村上隆 (alive) /
  Picasso (d.1973/until 2043) / Warhol (d.1987/until 2057) / Dali (d.1989/until 2059) /
  Miro (d.1983/until 2053) / Chagall (d.1985/until 2055) / Banksy (alive) / Hirst (alive)
  IF: source describes unauthorized reproduction/print of a protected artist's work
  THEN: CANNOT_LIST — copyright infringement
  NOTE: Resale of genuine original print = ALLOWED. Reproduction without license = CANNOT_LIST.

Trigger 3: CITES_MATERIAL
Condition: Source mentions ivory frame inlays (象牙額装飾) / tortoiseshell fittings (鼈甲)/
  rhinoceros horn (犀角) in frame or box fittings
Action: CANNOT_LIST — CITES Appendix I material
WARNING: Never write the word 'ivory' in any output field. Use 'faux ivory' only if source
  explicitly states synthetic/imitation material.

--- Safety gates (listing allowed with conditions) ---

Gate A: NEEDS_EXPORT_CLEARANCE
Condition: Print was manufactured before 1926 (over 100 years old) AND shipped from Japan
Certificate: 古美術品輸出鑑査証明書 (Agency for Cultural Affairs / 文化庁)
Processing: 10 business days, valid 1 year, single-use
Scope: All Edo-period Ukiyo-e (pre-1868) and Meiji-period prints (pre-1926) qualify
Customs: HTS 9702.10 (100+ year original print, duty 0%)
Action: Add to Description — "Export certificate required for international shipping."

Gate B: NEEDS_AUTHENTICATION_EVIDENCE
Condition: Source claims attribution to a Tier 1 artist (§9/§10) without providing:
  (a) publisher seal (版元印) / carver seal (彫師印) from verified publisher
  (b) edition number with physical pencil notation
  (c) expert appraisal / certificate of authenticity (COA) from recognized body
  (d) catalog raisonné reference number
Action: Do NOT include artist name in Title as definitive attribution.
  Use "Attributed to [Artist]" or "After [Artist] Style" in Description only.
  Remove "Signed by [Artist]" / "By [Artist]" claims without evidence.

Gate C: NEEDS_DISCLOSURE
Condition: Print is a later impression / reproduction / Giclee / Heisei/Reiwa edition
Mandatory disclosures:
  - 後刷 / 後摺り (Atozuri / Late Impression): state "Later Impression" in Title AND Description
  - 平成版 (Heisei Edition, post-1988 Watanabe): state "Heisei Edition" — IS: Vintage Reproduction
  - 令和版 (Reiwa Edition, post-2019 Watanabe): state "Reiwa Edition" — IS: Contemporary
  - 現代復刻 / 複製 (Modern Reproduction): state "Modern Reproduction" / "Reproduction" in Title
  - Giclee / ジクレー / デジタル複製: state "Giclee Print" or "Digital Reproduction" in Title
  - 遺作版 (Posthumous Edition): state "Posthumous Edition" in Description
Action: Omitting these disclosures violates eBay Item Not as Described policy.

Gate D: CANNOT_CLAIM_AS_STATED
Condition: Source lacks physical evidence for these claims:
  - Artist attribution without verifiable evidence → remove from Title
  - "Original Print" / "First Edition" without publisher record / edition watermark → remove
  - "Signed" / "Hand-Signed" without visible signature in source → remove
  - Edition number (e.g., 15/100) without pencil notation visible in source → remove
  - "Rare" / "Unique" / "Only One" without explicit source confirmation → remove (Rule 2 Cat B)

Gate E: HANGA_CLEAR
Condition: None of the above triggers or gates apply
Action: Proceed to generate listing

§4 Rule 2: NO PADDING — SOURCE-BASED CONDITIONAL ALLOWANCE
========================================

Category A — ABSOLUTE BAN (even if factual):
Subjective hype: AMAZING / FANTASTIC / INCREDIBLE / AWESOME / MUST HAVE / BEST / PERFECT
Empty superlatives: ULTIMATE / PREMIUM / LUXURY

ABSOLUTE BAN — eBay rights protection (NO exceptions):
AUTHENTIC / OFFICIAL

Source-transcription only — BANNED standalone, allowed ONLY when transcribing exact
material or accessory names directly from source text:
GENUINE — e.g., "Genuine Washi" (material from source)
ORIGINAL — e.g., "Original Box" (accessory from source); "Original Print" ONLY with
  verifiable evidence of direct printing from original matrix (§2 Gate D)

Category B — CONDITIONAL ALLOWANCE (only when source explicitly confirms):
- RARE — Only if source states rarity explicitly (limited edition count, documented scarcity)
- LIMITED / LIMITED EDITION — Only with edition number (e.g., 15/100) visible in source
- VINTAGE — Only if source confirms era with decade markers or explicit vintage statement
- SIGNED — Only if signature visually confirmed in source images or explicitly stated
- FIRST EDITION — Only with publisher record or edition watermark comparison evidence
- ORIGINAL PRINT — Only with evidence of direct printing from original matrix (§2 Gate D)

Category C — ALWAYS ALLOWED (factual attributes from source):
Print type, artist name (authenticated), medium, paper type, dimensions, year/era stated,
publisher stated, edition number stated, subject, condition details, framing status.

Rule: Source does not provide evidence → Do NOT include Category B word.
Guessing ("this looks rare"), inferring from comparable items, or using general knowledge
about an artist's market value = FORBIDDEN.

§11 EDITION DICTIONARY
========================================

- オリジナル版 / 原版 → Original Print
  Direct print from original matrix (block/plate/stone). Highest value.
  CANNOT CLAIM without: publisher record / edition proof / expert authentication.

- 初版 / 第1刷 → First Edition / First Printing / Early Impression
  First state of the edition. CANNOT CLAIM without publisher record or edition comparison.

- 後刷 / 後摺り → Later Impression / Atozuri
  Printed from original blocks at a later date. NEEDS_DISCLOSURE in Title AND Description.
  Characteristics: worn lines, faded colors, registration drift.

- 限定版 (例: 15/100) → Limited Edition (e.g., 15/100)
  Pencil notation of edition number required. Edition number without physical pencil =
  CANNOT_CLAIM_AS_STATED.

- AP / アーティストプルーフ → Artist's Proof (AP)
  Typically ≤10% of edition. Pencil "AP" notation required for claim.

- EA (Epreuve d'artiste) → Artist's Proof (EA) [French equivalent of AP]

- PP → Printer's Proof (PP) [printer's retain copy; "PP" notation required]

- HC → Hors Commerce (HC) [non-commercial use; gallery/museum copies]

- TP → Trial Proof (TP) [test pull before edition printing; can be highly collectible]

- BAT → Bon a Tirer (BAT) [final approval proof that sets the standard for edition printing]

- 遺作版 / 死後版 → Posthumous Edition [published after artist's death; NEEDS_DISCLOSURE]

- ファクシミリ版 → Facsimile Edition [photographic reproduction; NEEDS_DISCLOSURE]

- オープンエディション → Open Edition [no quantity limit; common in reproductions]

CANNOT_CLAIM rules:
- "Signed" requires visible signature in source images or explicit statement
- "Hand-Signed" requires pencil/pen signature (stamp/seal ≠ hand-signed)
- "First Edition" requires edition watermark comparison or publisher record
- "Original" requires evidence of direct printing from original matrix
- Numbered (e.g., 15/100) requires visible pencil notation in source

========================================
§17 TITLE
========================================

HARD LIMIT: 80 chars maximum. eBay rejects titles exceeding 80 chars.
GOAL: Maximize SEO discoverability by packing every source-derived fact
(artist, print type, technique, series, edition, subject, era, publisher, signature status)
within the 80-char budget. Unused characters = lost search visibility.
Shorter title is acceptable when source provides limited information.
Lying is worse than being short.

Allowed symbols: & / : - . , '
  Note: Apostrophe (') allowed for Artist's Proof / Printer's Proof / proper names.
Forbidden symbols: * $ ~ ^ @ ! ? ™ ( ) [ ] { } " _ % = | \ < > ;

SEO KEYWORD ORDER (front-load highest-value terms):
1. Artist name (authenticated, within first 30 chars) OR print type if artist unknown
2. Print type / technique (Woodblock Print / Ukiyo-e / Lithograph / Etching / Shin-hanga)
3. Series name if famous (Tokaido / 36 Views / Aspects of the Moon)
4. Edition / impression type (Limited Edition 15/100 / AP / Later Impression)
5. Subject / motif (Landscape / Snow Scene / Mt. Fuji / Bijin-ga / Dragon)
6. Signature status (Signed / Pencil Signed / Unsigned)
7. Publisher if notable (Watanabe / Mourlot)
8. Era / period if adds SEO value (Edo / Meiji / Taisho / Showa)
9. Reproduction status (MUST include if Later Impression / Reproduction / Giclee)

ARTIST NAME RULE:
- Use internationally recognized romanized spelling for known artists
- Hepburn romanization for Japanese names: Hiroshige / Hasui Kawase / Munakata Shiko
- If attribution unverified: "After Hiroshige" / "Attributed to Hasui" (not "By Hiroshige")
- If artist unknown: lead with print type ("Japanese Woodblock Print Landscape Edo")
- If CANNOT_CLAIM_AS_STATED: omit artist name from Title

DISCLOSURE RULE IN TITLE:
- Later Impression / Atozuri → MUST include "Later Impression" or "Reprint" in Title
- Reproduction / Heisei/Reiwa edition → MUST include "Reproduction" or era in Title
- Giclee → MUST include "Giclee Print" in Title

========================================
§18 DESCRIPTION
========================================

English ASCII only, <= 480 chars, single natural paragraph.

FIRST SENTENCE: Must state print type + artist + subject (or "unknown artist" if unverified).
Example: "Japanese woodblock print by Hiroshige depicting a snow scene from the
Fifty-three Stations of the Tokaido series..."

Include when source provides:
- Artist name (romanized) and attribution confidence level
- Print type / technique (specific: Mokuhanga / Lithograph / Etching)
- Series name (with English translation)
- Edition number and total (e.g., "no. 15 of 100")
- Impression type (Early Impression / Later Impression / Reproduction)
- Signature / seal details (pencil-signed / seal/rakkan / unsigned)
- Publisher and edition information (Watanabe 1920s edition)
- Paper type (Washi / Kozo / Arches)
- Dimensions: image size AND frame size in cm (and inches if stated)
- Framing status (Framed with acid-free mat / Unframed)
- Year / era of printing
- Provenance / COA reference (for Tier 1 artists)
- Export certificate note (for pre-1926 prints shipped from Japan — Gate A)
- ALL known defects (§15 DEFECT MAPPING) — mandatory

FIRST SENTENCE DEFECT INTEGRATION:
If defects exist, weave them naturally: "...in good used condition with minor foxing to margins."
Do NOT create a separate "Defects:" list — integrate into the natural flow.

Exclude: shipping policies / seller opinions / self-promotion.
All measurements in cm (primary) followed by inches in parentheses if source provides both.
