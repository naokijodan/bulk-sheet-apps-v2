# PC BODY (COMPUTERS) - Translation Rules

## GOALS
- Convert Japanese computer listings to English eBay-optimized titles
- Detect the correct subtype from the source: Laptop / Desktop / Tablet
- Extract subtype-specific Item Specifics without padding or invention
- Preserve accessories, defects, missing parts, and working status exactly as stated

## ABSOLUTE RULES (MUST NOT)
- TRUTH FIRST: Do not fabricate any specification, accessory, condition, grade, or model detail
- NO PADDING (Category A): Do not add Authentic / Genuine / Original / Verified / Premium unless source explicitly states it
- NO PADDING (Category B): New / Sealed / Working / Tested / Refurbished / Upgraded only when source confirms it
- NO PADDING (Category C): Descriptive facts are allowed only when they are supported by the source text
- SOURCE-ONLY: Every output detail must trace to the Japanese source
- NO CJK: English output only in the final result, ASCII only
- DEFECT FIRST: If the source says junk, broken, missing parts, battery issue, screen issue, or activation lock, keep it
- DO NOT FORCE FIELDS: Missing information should stay NA rather than guessed

## LEGITIMATE KEYWORDS (only if in source)
- New / Unused / Open Box / Used
- Tested / Working / Fully Working
- For parts or not working / Junk
- SSD upgraded / RAM upgraded / Battery replaced
- Japanese Layout (JIS) / US Layout (ANSI)
- Japan Import / Japanese Market
- Wi-Fi / Wi-Fi + 4G / Wi-Fi + 5G / Cellular
- Touchscreen / Backlit Keyboard / Webcam / Fingerprint Reader

## TYPE DETECTION (routes IS output)
- ノートパソコン / ノートPC / ラップトップ / MacBook / ThinkPad / Chromebook / Lets note / VAIO / dynabook / LIFEBOOK / LAVIE / Ultrabook -> Type: Laptop
- デスクトップ / デスクトップPC / iMac / Mac mini / Mac Pro / NUC / ミニPC / 一体型PC / オールインワン / ゲーミングPC / 自作PC / BTO -> Type: Desktop
- タブレット / iPad / iPad Pro / iPad Air / iPad mini / Surface Pro / Surface Go / Android tablet / Galaxy Tab -> Type: Tablet
- If subtype is unclear, prefer the most conservative source-based choice. Do not force Desktop or Tablet without evidence.

## SUBTYPE FIELD PRIORITY

- Brand / Model / Type / Series
- Processor / Processor Speed / RAM Size

- Operating System / Screen Size / Screen Resolution

### Desktop
- Brand / Model / Type
- Processor / RAM Size / SSD Capacity / Hard Drive Capacity
- Operating System / Form Factor / GPU
- Color / Connectivity / Year Manufactured

### Tablet
- Brand / Model / Storage Capacity / Operating System
- Screen Size / Internet Connectivity / Color
- RAM Size / Processor / Battery Run Time / Connectivity / Year Manufactured

## TITLE COMPOSITION GUIDANCE (examples, not mandatory)
- Compose naturally from the strongest source-backed details
- Prefer Brand + Model + subtype-relevant specs + condition cue when space allows
- Do not follow a rigid title formula if the source is sparse
- Skip missing fields instead of padding
- Keep subtype identity clear early in the title when helpful

Examples:
- Apple MacBook Pro 14 M3 Pro 18GB 512GB SSD Japanese Keyboard
- Dell OptiPlex 7090 Desktop i7 16GB 512GB SSD Windows 11 Used
- Apple iPad Pro 12.9 256GB Wi-Fi Space Gray iPadOS Used
- Panasonic Lets note CF-SV8 i5 8GB 256GB SSD Laptop Junk

## DESCRIPTION RULES
- Single paragraph only
- ASCII only
- Keep it factual and compact
- Mention accessories only when present in source
- Mention defects clearly when present in source
- Do not claim battery health, exact benchmark, release year, or upgrade history without evidence
