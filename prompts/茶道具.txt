SECTION 3: ABSOLUTE RULES (apply to all output)
========================================

Rule 1 — TRUTH FIRST
Every fact in Title / Description / ProductName MUST be derivable from source (title_jp, description_jp, tag_jp). Do NOT add details from training knowledge. Do NOT complete partial claims.

Rule 2 — NO PADDING (source-based conditional allowance)
Category A — ABSOLUTE BAN (even if factual):
  Subjective hype: AMAZING / FANTASTIC / INCREDIBLE / AWESOME / MUST HAVE / BEST / PERFECT
  Empty superlatives: ULTIMATE / PREMIUM / LUXURY
  VeRO ban (no exceptions): AUTHENTIC / OFFICIAL
Category B — CONDITIONAL (only if source explicitly confirms):
  RARE / LIMITED / VINTAGE / EXCLUSIVE / COLLECTIBLE / MINT
  GENUINE / ORIGINAL — transcribe only if source contains the exact term as material or accessory name
Category C — ALWAYS ALLOWED: material, era, size, condition, maker when source-supported

Rule 3 — NO REPETITION
No token repeated more than once in Title.

Rule 4 — TITLE BEGINS WITH JAPANESE IDENTIFIER
Title MUST begin with a Japanese-origin utensil type, material, maker, or technique term (Natsume / Chashaku / Lacquer / Urushi / Maki-e / Shifuku / Furo / Tana etc.). The generic word "Japanese" alone as the first and only keyword is insufficient. Japanese-origin terms front-load the highest search value.

Rule 5 — SOURCE-ONLY ATTRIBUTION
Authority claims (maker / tea master / inscription / provenance) are transcribed ONLY from source evidence. AI MUST NOT infer, guess, or complete claims beyond what the source states.

-- BOX HIERARCHY --
共箱 (Tomobako) — original box made by the maker; highest maker evidence
識箱 / 鑑定箱 (Shikibako) — appraisal or authentication box by third party; not maker's box
合わせ箱 (Awasebako) — replacement box assembled later; note in description
箱なし — no box included

-- AUTHORITY INSCRIPTION TYPES --
大徳寺箱書き (Daitokuji hakogaki) — highest Buddhist authority inscription; confirms temple connection ONLY; does NOT confirm maker identity
茶人箱書き (Chajin hakogaki) — inscription on box by a named tea master (利休 / 遠州 / 不昧公 / 不白 etc.)
極め書き (Kiwamegaki) — expert appraisal inscription
書付 (Kakitsuke) — written notes or attribution on box
折紙 (Origami) — formal authentication document (historical usage)
由緒書 (Yuisho-gaki) — provenance document
鑑定書 (Kanteisho) — formal certificate of authentication
売立目録 / 競売記録 — historical auction catalog record
Gallery receipt / auction provenance — institution-attributable written record

CRITICAL: Daitokuji hakogaki is NOT a maker's signature. Do NOT infer maker from box inscription alone.

-- 7a. Maker Expression — 6 levels (source evidence required) --

Level 1: By [Maker]
  Conditions: 共箱 AND (鑑定書 OR gallery receipt OR auction provenance OR 購入書類)
  VERY HIGH Senke Jusshoku current generation: requires two independent evidence forms
  Example: Japanese Lacquer Natsume By Nakamura Sotetsu Tomobako

Level 2: [Maker] signed / [Maker] marked
  Conditions: physical signature or impressed mark on object confirmed in source; no document required
  Example: Japanese Chashaku Signed Kuroda Shogen with Tube

Level 3: Attributed to [Maker]
  Conditions: source explicitly uses attribution-language (attributed to / と伝わる / 伝 / 推定); no definitive evidence
  Example: Japanese Tana Attributed to Hiki Ikkan Lacquer

Level 4: School of [Maker]
  Conditions: source confirms workshop lineage; not the maker himself
  Example: Japanese Natsume School of Sotetsu

Level 5: In the manner of [Maker] / after [Maker]
  Conditions: style influence confirmed by source
  Example: Japanese Natsume In the Manner of Sotetsu

Level 6: Unknown maker / Unsigned
  Conditions: no maker evidence in source
  Example: Japanese Lacquer Natsume Unsigned Antique

-- 7b. Authority Inscription Expression — 6 levels (independent of maker evidence) --

A1: with Daitokuji hakogaki
  Source explicitly states 大徳寺箱書き or Daitokuji inscription
  Example: Japanese Natsume Maki-e with Daitokuji Hakogaki Box

A2: with Chajin hakogaki by [TeaMaster]
  Source names the tea master who wrote the box inscription
  Example: Japanese Lacquer Natsume with Chajin Hakogaki by Fumai

A3: with hakogaki
  General box inscription confirmed; source does not name the author
  Example: Japanese Shifuku with Hakogaki Box

A4: with chashaku mei
  Tea scoop carries a name inscription; note if tea master named it
  Example: Japanese Chashaku with Mei Susutake Antique

A5: with kanteisho / with written authentication
  Formal authentication certificate included
  Example: Japanese Natsume By Sotetsu with Kanteisho

A6: no inscription
  Source explicitly confirms absence of inscription
  Example: Japanese Lacquer Natsume No Inscription Antique

-- 7c. Tea Master Attribution Expression — 9 levels (independent dimension) --

T1: made by [TeaMaster]
  Source states 作 or 制作 AND provides TWO evidence forms: (茶人箱書き or 大徳寺箱書き) + (鑑定書 or gallery receipt)
  VERY HIGH masters (利休 / 紹鴎 / 織部 / 遠州 / 石州 / 不昧公): T1 requires double evidence — AI MUST NOT apply T1 on single source alone

T2: inscribed by [TeaMaster]
  Tea master wrote the box inscription or chashaku mei; source confirms authorship of inscription

T3: named by [TeaMaster]
  Tea master gave the object its poetic name (銘); source states this explicitly

T4: owned by [TeaMaster] / from collection of [TeaMaster]
  Provenance ownership; source provides 由緒書 or 図録 or competitive auction record

T5: attributed to [TeaMaster]
  Source explicitly uses attribution-language (attributed to / 伝 / と伝わる); no definitive evidence
  CRITICAL: For VERY HIGH masters (利休 / 紹鴎 / 織部 / 遠州 / 石州 / 不昧公), T5 without double evidence → NEEDS_AUTHENTICATION_EVIDENCE (not LISTABLE). T5 LISTABLE only applies to non-VERY-HIGH masters.

T6: in the style of [TeaMaster]
  Style reference only; no specific authorship or ownership claim

T7: associated with [TeaMaster]
  Contextual relationship stated in source (used at a ceremony of / related to collection of)

T8: school of [TeaMaster] / [TeaMaster] tradition
  Lineage school confirmed; not the master himself

T9: copy of / utsushi after [TeaMaster]
  Acknowledged reproduction in the style of the master; source states 写し or utsushi

-- 7d. Chashaku Inscription Claim — independent dimension for 茶杓 --

茶杓 has multiple compound attribution dimensions: maker of scoop + tube maker + inscription on tube + name given to scoop + who gave the name. Treat each separately.

Permitted forms:

- Chashaku named "[Mei]" by [TeaMaster] — name given by master
