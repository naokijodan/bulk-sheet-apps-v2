SECTION 4: EVIDENCE GATE
========================================

Only claim attributes with documented evidence in source text, photos, or documents.
Never attribute maker, era, or authenticity from visual impression alone.

Source evidence tiers (highest to lowest confidence):
  1. Published catalog documentation (図録掲載) with this specific item
  2. Third-party expert certificate (鑑定書) naming maker
  3. Original box (共箱) with maker inscription (箱書き)
  4. Visible seal or impressed mark on item body (落款 / 印 / 刻印)
  5. Seller attribution statement citing specific source (text, receipt, appraisal)

PROHIBITED inferences:
  - Attributing maker from visual style alone (glaze color, shape similarity)
  - Inferring era from design motif without explicit source marker
  - Claiming "authentic" without evidence-tier support
  - Assuming provenance from regional origin alone

RULE 2: NO PADDING — SOURCE-BASED CONDITIONAL ALLOWANCE
========================================

Category A — ABSOLUTE BAN (even if factual):
  Subjective hype: AMAZING, FANTASTIC, INCREDIBLE, AWESOME, MUST HAVE, BEST, PERFECT
  Empty superlatives: ULTIMATE, PREMIUM, LUXURY, SUPERIOR, FINEST
  eBay rights protection (NO exceptions): AUTHENTIC, OFFICIAL

  Source-transcription only (BANNED standalone):
  GENUINE — allowed ONLY as material name (e.g., "Genuine Leather")
  ORIGINAL — allowed ONLY as accessory name (e.g., "Original Box")

Category B — CONDITIONAL ALLOWANCE (only if source contains explicit evidence):
  RARE — only if source states specific rarity (limited production, one of N made)
  LIMITED / LIMITED EDITION — only if source confirms limited production count
  VINTAGE — only if source confirms era with decade or year markers
  EXCLUSIVE — only if source confirms exclusive distribution
  COLLECTIBLE — only if source explicitly marks item as collectible
  MINT — only if source states unused + original box + never used

Category C — ALWAYS ALLOWED (factual source attributes):
  Material, maker/kiln (with evidence gate), shape, size, era (evidenced),
  origin, condition, included accessories, dimensions
