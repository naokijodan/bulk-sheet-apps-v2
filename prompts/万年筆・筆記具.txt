5. Route non-pen items to correct prompt per BOUNDARY RULE (mandatory STOP)

===== CONDITION TERMS (2 AXES) =====

AXIS 1 — FUNCTIONAL CONDITION (writing performance):
Writes Smoothly — tested, ink flows well without issues
Ink Flow Good — good flow, minor occasional start issues acceptable
Hard Starting — requires extended warming before ink flows
Skipping — intermittent ink gaps during writing
Filling Mechanism Working — piston / vacuum / lever etc. fills correctly
Filling Issue — mechanism stiff, incomplete fill, or not filling
Tested-Working — basic function verified, writes as expected
For Parts — non-functional; may be incomplete or damaged

AXIS 2 — COSMETIC CONDITION:
Mint — no visible use, no wear marks, pristine
Excellent — minimal signs of use, no notable imperfections
VG (Very Good) — light normal wear, no significant defects
Good — visible use wear, minor scratches acceptable
Fair — notable cosmetic damage affecting presentation
Poor — significant damage or major cosmetic issues
Describe specifically: Scratches / Plating Wear / Cap/Barrel/Nib Appearance /
Maki-e Chipping / Body Discoloration / Celluloid Yellowing / Ebonite Oxidation

PROHIBITED CONDITION TERMS for Writing Instruments:
TCG grading terms PROHIBITED: LP (Lightly Played) / MP / HP / DMG / PSA / BGS / Graded slab
These are trading card terms and must never be used for pens.

===== MAKI-E / PREMIUM MATERIAL SPECIAL RULES =====

MAKI-E (蒔絵) RULE:
- Include "Maki-e" in Title and Description ONLY when source explicitly states:
  蒔絵 / 漆 / 蒔絵師名 (artist name) / Maki-e / Urushi
- DO NOT identify Maki-e from visual description (e.g. "has a floral pattern" — not sufficient)
- DO NOT assume Maki-e from model name alone (e.g. Namiki does not always = Maki-e)
- When Maki-e confirmed: include in Title before Nib info (high SEO + high value)
  Example: Pilot Namiki Maki-e Crane Fountain Pen 18K M Nib Japan

STERLING SILVER / SOLID GOLD / PLATINUM TRIM RULE:
- Include ONLY when source explicitly states 純銀 / 純金 / プラチナ装飾 / sterling / solid gold
- DO NOT infer from model number alone
- When confirmed: include in Title after Brand + Model
  Example: Montblanc 149 Sterling Silver Fountain Pen 18K M Nib

LIMITED EDITION / PATRON OF ART / HERITAGE RULE:
- Include ONLY when source explicitly states limited edition name or edition designation
- DO NOT infer edition from catalog number alone (e.g. year-suffix models)
- When confirmed: include in Title
  Example: Montblanc Writers Edition Hemingway Fountain Pen 18K M Nib 1992
