# ELECTRONIC DICTIONARIES - Translation Rules

## GOALS
- Convert Japanese electronic dictionary listings to English eBay optimized titles
- Extract Brand / Model / Type / Dictionary Language / Number of Dictionaries / Screen Size / Features / Power Source / Country of Origin / Color / Condition
- Preserve accessories, defects, battery issues, touch issues, and missing parts exactly as stated

## ABSOLUTE RULES
- TRUTH FIRST: Use only facts supported by the source text
- NO PADDING Category A: Do not add Amazing / Fantastic / Incredible / Best / Perfect / Premium / Luxury
- NO PADDING Category A: AUTHENTIC and OFFICIAL are banned unless part of a direct accessory name in source
- NO PADDING Category B: New / Sealed / Working / Tested / Complete / Battery Included / Adapter Included only when source confirms
- NO PADDING Category C: Brand / Model / Language / Screen Size / Features / Power Source / Color / Country / Condition are allowed only when source states them
- SOURCE-ONLY: Do not infer from general product knowledge beyond source keywords and obvious keyword mapping below
- NO CJK: Final output must be English ASCII only
- DO NOT FORCE FIELDS: If the source does not support a field, write NA
- DEFECT FIRST: If the source says junk, broken, battery leak, touch failure, missing stylus, or no power, keep it

## LEGITIMATE KEYWORDS
- New / Used / Open Box / Unused
- Working / Tested / Powers On
- Battery Included / AC Adapter Included / USB Cable Included
- Touch Panel / Voice Recognition / Camera / Handwriting Input
- Japan Import / Japanese Market
- Multilingual / English Japanese / Japanese English

## TITLE COMPOSITION GUIDANCE
- Build the title naturally from the strongest source backed details
- No fixed title formula
- Prefer Brand + Model + product identity + language or feature + condition cue when space allows
- Skip missing fields instead of padding
- Use Electronic Dictionary in the title when the item is clearly an electronic dictionary

Examples:
- Casio EX-word XD-SX8900 Electronic Dictionary Japanese English 200 Dictionaries
- Sharp Brain PW-A7000 Electronic Dictionary English Japanese Multilingual
- Canon wordtank IDP-920D Electronic Dictionary Handheld Japan Import
- Seiko IC Dictionary SR Series Electronic Dictionary Used

## DESCRIPTION RULES
- Single paragraph only
- ASCII only
- Keep it factual and compact
- Mention included accessories only when present in source
- Mention defects clearly when present in source
- Do not claim exact release year, battery life, or dictionary count without evidence
