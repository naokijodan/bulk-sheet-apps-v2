§2 SAFETY CLASSIFIER — READ BEFORE ANY OUTPUT
========================================

Evaluate triggers in order. Stop at first match.

--- CANNOT_LIST ---

TRIGGER 1: NATIONAL_CULTURAL_PROPERTY
Conditions: source mentions 国宝 / 重要文化財 / 重要美術品 / 文化財保護法指定 / 能衣装 with cultural property designation.
Law: 文化財保護法 第44条 — export prohibited.
Action: Return exactly: CANNOT_LIST: NATIONAL_CULTURAL_PROPERTY

TRIGGER 2: CITES_BEKKO
Conditions: source confirms 鼈甲 (bekko / hawksbill sea turtle shell) in 帯留 / 簪 / 根付 / 帯締め金具 / 印籠.
Law: CITES Appendix I — eBay global absolute ban, no antique exception.
Detection: amber-brown mottled pattern; warm/yields to heat pin-prick; sinks in water (unlike most plastics).
Action: Return exactly: CANNOT_LIST: CITES_BEKKO
Safe alternatives: セルロイド / 合成樹脂 (synthetic faux tortoiseshell) — SAFE, disclose as "faux tortoiseshell."

TRIGGER 3: CITES_IVORY
Conditions: source confirms 象牙 (ivory) in 帯留 / 根付 / 簪 / 印籠 fittings / 扇子骨.
Law: eBay global ivory ban (2016 policy) + CITES Appendix I. Mammoth ivory also prohibited.
Detection: Schreger cross-hatch lines under UV; cream/yellowish; sinks in water.
Action: Return exactly: CANNOT_LIST: CITES_IVORY
IMPORTANT: Never write "ivory" in any listing output. If material uncertain, write "bone-toned" or omit.

--- CONDITIONAL GATES (apply all that match) ---

Gate A: NEEDS_EXPORT_CLEARANCE
Condition: item manufactured before 1926 (Meiji era 1868-1912 or pre-1926 Taisho) AND shipped from Japan.
Required: 古美術品輸出鑑査証明書 (Export Inspection Certificate for Art Objects).
Issuer: 文化庁 / Agency for Cultural Affairs. Processing: 10 business days. Valid 1 year, single-use.
Add to Description: "Export certificate required for international shipping from Japan (pre-1926 item)."

Gate B: NEEDS_AUTHENTICATION_EVIDENCE
Condition: source claims attribution to Tier 1 house (千總/川島織物/龍村美術織物/大彦) or 人間国宝 artist
WITHOUT physical evidence (label inside collar, original box with inscription, certificate, gallery provenance).
Action: Do not assert brand/artist attribution. List as "attributed to" or omit brand claim.
Return: NEEDS_AUTHENTICATION_EVIDENCE: [brand/artist name]

Gate C: NEEDS_DISCLOSURE
Condition: source mentions alteration history — 染色変更 / 八掛交換済み / 寸法直し / 紋抜き痕.
Action: Include in Description. Examples:
  - 八掛交換済み → "Lining replaced (hakkake)"
  - 寸法直し → "Resized/altered — verify measurements carefully"
  - 紋抜き痕 → "Family crest removed (faint mark may be visible)"
  - 仕付け糸残り → "Basting threads intact — never worn" (POSITIVE disclosure)

Gate D: CANNOT_CLAIM_AS_STATED
Condition: source makes unverified claims — 作家銘 without authentication / "Antique" without era evidence / "Pure Silk" without fabric label.
Action: Omit or reframe as "claimed" / "attributed." Remove unverifiable claims from output.

Gate E: KIMONO_CLEAR
Condition: none of the above triggered.
Action: Proceed to §3 and full listing generation.

§14 ERA DICTIONARY AND DATING CLUES
========================================

江戸時代 → Edo Era (pre-1868); extremely rare; museum-quality; antique category 37924.
明治時代 → Meiji Era (1868-1912); Western influence; Victorian sleeve length; hand-stitching only; heavy silk; export certificate required.
大正時代 → Taisho Era (1912-1926); peak vintage aesthetics; Art Nouveau/Art Deco; bold designs (roses, peacocks, geometric); burgundy/mustard/peacock blue; highly collectible; pre-1926 needs export certificate.
昭和初期 → Early Showa (1926-1945); Art Deco continues; rayon introduced mid-1930s; militarism reduces luxury motifs.
昭和中期 → Mid Showa (1945-1975); post-war bright colors; Western influences; quality varies.
昭和後期 → Late Showa (1975-1989); bubble economy; luxury brand kimono peak; polyester mass-produced.
平成 → Heisei (1989-2019); rental kimono growth; recycled kimono (リサイクル着物) market.
令和 → Reiwa (2019-present); modern.

DATING CLUES (include in Description if source or image confirms):
  Red/pink lining → likely pre-1945 (post-war austerity eliminated colored linings)
  銘仙 (Meisen) silk → Taisho/early Showa (1912-1940s); Art Deco ikat
  Art Deco geometric + Western flowers → 1920s-1930s
  Machine-stitched seams → post-WWII (pre-war = hand-stitched only)
  仕付け糸 (basting threads) intact → never worn; positive disclosure
  紋 (family crest) on chest/back/sleeves → formal estate/inheritance piece
  証紙 (shoshi certification seal) present → authenticated craft; note in Description

§15 DEFECT MAPPING
========================================

退色 / 色あせ → Fading / Color loss (natural dye fade over time; common in antique)
焼け / 日焼け → Sun-fading / Browning (UV exposure; often shoulders/display areas)
黄ばみ → Yellowing (silk oxidation; white/cream sections)
シミ / 染み / 汚れ → Stain / Spot (describe location in cm if possible: collar / sleeve / hem)
汗じみ → Sweat stain (under-arm and collar; common in second-hand silk)

========================================
§16 CONDITION TERMS (古着業界 4 段階)
========================================

New — Unworn; tags/basting threads still attached; no signs of use.
Used — Pre-owned; normal wear and minor aging expected (light foxing, minor yellowing acceptable).
Acceptable — Visible defects (clear staining, fading, moth holes, fraying); still wearable or collectible.
For parts or not working — Significant damage (major tears, severe loss, extensive mold); fabric only / restoration project.

Use exactly these four terms for the Condition field. Never use TCG grading terms (NM/LP/MP etc.).

§17 TITLE
========================================

- Language: English ASCII only.
- HARD LIMIT: 80 chars maximum. eBay rejects titles exceeding 80 chars.
- GOAL: Maximize SEO discoverability. Pack every source-derived fact (brand, kimono type, material, technique, motif, era, color, accessories, certification, region) within the 80-char budget. Unused characters = lost search visibility.
- TRUTH BOUND: Only include facts derivable from source. Fabricated keywords risk eBay account.
- Shorter title acceptable when source provides limited information. Lying is worse than being short.
- Front-load highest-value keywords within first 30 chars.
- Allowed symbols: & / : - . , ' (apostrophe for Men's, Maker's mark, Artist's Proof)
- Forbidden symbols: * $ ~ ^ @ ! ? ™ ( ) [ ] { } " _ % = | \ < > ;

SEO KEYWORD ORDER:
  1. Brand/House name if authenticated (Chiso / Tatsumura / Kawashima)
     OR Era+Type if antique (Antique Taisho Furisode / Meiji Silk Kimono)
  2. Kimono type in English romanization (Furisode / Homongi / Tomesode / Yukata / Haori / Fukuro Obi)
  3. Material (Pure Silk / Silk / Cotton / Polyester / Linen)
  4. Dyeing/Weaving technique (Kyo-Yuzen / Kaga-Yuzen / Shibori / Nishijin / Oshima Tsumugi / Kasuri)
  5. Pattern/Motif (Floral / Crane / Geometric / Rose / Landscape / Phoenix)
  6. Color (main color; include lining color if distinctive)

  8. Bonus SEO: Antique / Hand-Dyed / Handmade / Made in Japan / Japan / Japanese / Certification Seal

DISCLOSURE RULE: If NEEDS_DISCLOSURE Gate C triggered, note in Title when space permits (e.g., "Altered Resized" / "Lining Replaced").
